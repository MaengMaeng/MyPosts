## 개요

마크업 노트



## 개발 스택

- React
  - Hook, functional component
- redux
- mongodb
- typescript
- express
- sass



## 프로젝트 내용

- 마크업 노트
  - 상단 바
    - 로그인
    - 메뉴
      - 글
      - 검색
  - 메인 페이지
    - 글 읽기/쓰기/수정/삭제
  - 하단 바
    - 만든 사람



## 고려할 점

- 단위(unit) 테스트
  - Mocha, Chai, Jest, Selenium 등
- 배포



## 실행

```
npm start
```
